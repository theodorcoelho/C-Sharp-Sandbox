# C-Sharp-Sandbox

**C-Sharp-Sandbox:** repositório criado para explorar e praticar conceitos fundamentais da linguagem C#.  

## Sobre o Projeto
Este repositório contém códigos e exemplos práticos utilizados para aprender e testar funcionalidades do C#.  
O objetivo é criar um ambiente dinâmico para experimentação, sem a necessidade de seguir uma estrutura rígida de projeto.

## Tópicos Praticados
Durante o desenvolvimento deste projeto, estou estudando e praticando os seguintes conceitos:

- **Variáveis e Tipos de Dados**
  - Conceito de variável
  - Declaração de variáveis e tipos
  - Outras formas de declaração
  - Constantes
  - Capturando entrada do usuário

- **Operadores**
  - Operadores aritméticos
  - Operadores lógicos (teoria e prática)

- **Estruturas Condicionais**
  - Condicionais (teoria e prática)
  - Switch
  - Enum
  - Enum com Switch

- **Funções e Escopo**
  - Funções
  - Retorno de funções
  - Escopo

- **Arrays e Laços de Repetição**
  - Arrays
  - While
  - Do While
  - Foreach
  - For
